<b>README</b>

This README describes steps are necessary to get the application up and running.

You need to have:
* Ruby version 1.9.3 http://railsinstaller.org/

Actions to run:
* Clone repository `git clone`

* Install dependencies `bundle install` 

* Setup database `rake db:create db:migrate db:seed`

* Run server `rails server`

* Go to page in your browser `localhost:3000`


Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
